## Still Needed

### App Store Badge
Download the official "Download on the App Store" badge:
1. Visit: https://developer.apple.com/app-store/marketing/guidelines/#downloadOnAppstore
2. Download the SVG file
3. Save as: `app-store-badge.svg` in this folder

### Favicons
Generate favicons from your app icon using:
1. Visit: https://realfavicongenerator.net/
2. Upload your StarWell app icon (1024x1024)
3. Generate all sizes
4. Download and extract to this folder

Required favicon files:
- `favicon-32x32.png`
- `favicon-16x16.png`
- `apple-touch-icon.png` (180x180)

## Image Optimization

Before uploading, optimize all images:
- Use https://tinypng.com/ or ImageOptim (Mac)
- Target size: ~200-300KB per screenshot
- Keep quality high but file size reasonable
